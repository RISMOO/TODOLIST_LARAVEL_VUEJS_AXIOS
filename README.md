

## Todolist Réalisé en Laravel et vue js avec requetes Axios


***
## START LARAVEL
     composer create-project laravel/laravel Projet
    

## RUN
     cd Projet
     php artisan serve


***
## START VUEJS
    composer require laravel/ui
    php artisan ui vue
    npm install
    npm run dev
